    self-out

    Selfout is a mobile application build using flutter,node js and mongo db. It can be used by costumers to checkout items they want to purchase in a shop by scanning the bar codes by themselves.

    Customers can choose the shop of their choice. It will show all the available items in the store. The customer can scan the barcode of the item using the app, which will add the product into the virtual cart. The user can then checkout the items after payment.

    You can download the app here. https://drive.google.com/file/d/13y5AxrqQNnhvdXcW7p5jP-Py6lVQv6I-/view?usp=sharing
